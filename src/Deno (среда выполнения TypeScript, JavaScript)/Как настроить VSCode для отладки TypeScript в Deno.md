# Как настроить VSCode для отладки TypeScript в Deno?

Настройка VSCode для отладки TypeScript в Deno — довольно простой процесс, особенно если у вас уже установлен Deno и расширение Deno для VSCode.

Вот пошаговая инструкция:

**1. Установка расширения Deno для VSCode:**

- Откройте VSCode и перейдите на вкладку «Расширения» (Extensions) (Ctrl+Shift+X или Cmd+Shift+X).
- Найдите «Deno» и установите официальное расширение от Deno Land.

**2. Настройка файла `launch.json`:**

- Перейдите на вкладку «Запуск и отладка» (Run and Debug) (Ctrl+Shift+D или Cmd+Shift+D).
- Нажмите «создать файл launch.json» (create a launch.json file).
- Выберите «Deno».
- VSCode автоматически создаст файл `launch.json` с базовой конфигурацией для отладки Deno.
- Если у вас уже есть файл `launch.json`, добавьте в него следующую конфигурацию:

```json
{
	"version": "0.2.0",
	"configurations": [
		{
			"request": "launch",
			"name": "Запуск файла Deno",
			"type": "node-deno",
			"program": "${file}",
			"cwd": "${workspaceFolder}",
			"attach": false,
			"inspect": false,
			"justMyCode": true,
			"outputCapture": "std",
			"runtimeExecutable": "deno",
			"runtimeArgs": ["run", "--inspect-brk", "-A"],
			"port": 9229
		}
	]
}
```

- `"program": "${file}"` указывает, что отладчик должен запустить текущий открытый файл.
- `"runtimeArgs": ["run", "--inspect-brk", "-A"]` указывает Deno запустить файл в режиме отладки и предоставить все разрешения.
- Если вам нужны не все разрешения, то замените "-A" на нужные, например: "--allow-read", "--allow-write" и т.д.

**3. Установка точек останова:**

- Откройте файл TypeScript, который вы хотите отладить.
- Щёлкните на левой панели редактора, чтобы установить точки останова (красные кружки).

**4. Запуск отладки:**

- Перейдите на вкладку «Запуск и отладка» (Run and Debug) (Ctrl+Shift+D или Cmd+Shift+D).
- Выберите конфигурацию «Запуск файла Deno» из выпадающего списка.
- Нажмите кнопку «Запустить отладку» (Start Debugging) (зелёная стрелка) или нажмите клавишу F5.

**5. Отладка:**

- Когда выполнение программы достигнет точки останова, отладчик остановится.
- Вы можете использовать панель отладки для пошагового выполнения кода, просмотра переменных и т. д.

**Дополнительные советы:**

- Если у вас возникли проблемы с отладкой, убедитесь, что у вас установлена последняя версия Deno и расширения Deno для VSCode.
- Проверьте, правильно ли настроены разрешения в файле `launch.json`.
- Используйте панель «Консоль отладки» (Debug Console) для просмотра сообщений отладки.

Надеюсь, эта инструкция поможет вам настроить отладку TypeScript в Deno в VSCode.
