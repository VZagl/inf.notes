## Основное описание

Context7 MCP — это инструмент, который помогает пользователю получать актуальную документацию и примеры кода прямо в процессе работы. Он интегрируется с различными клиентами Model Context Protocol (MCP), такими как GitHub Copilot, Cursor, Windsurf и другие.

### Зачем это нужно пользователю:

- **Актуальность данных**: документация и примеры кода обновляются в реальном времени.
- **Экономия времени**: вместо поиска документации вручную, Context7 MCP предоставляет её автоматически.
- **Точность**: исключаются ошибки, связанные с устаревшими или несуществующими API.
- **Удобство**: вся информация доступна прямо в редакторе кода, без необходимости переключения между вкладками.
- **Поддержка версий**: Context7 MCP учитывает версию библиотеки или фреймворка, чтобы предоставить релевантные примеры.

### Пример использования:

1. Пользователь пишет запрос в GitHub Copilot, например: "Создай проект Next.js с маршрутизацией".
2. Context7 MCP автоматически добавляет актуальную документацию и примеры кода в ответ Copilot.
3. Пользователь получает готовое решение, которое можно сразу использовать в проекте.

Context7 MCP делает процесс разработки более быстрым, точным и удобным.

### Как это работает:

Context7 MCP использует следующие механизмы для обеспечения актуальности данных:

1. **Интеграция с API**: Context7 MCP подключается к официальным API библиотек и фреймворков, чтобы получать актуальную информацию о версиях, функциях и примерах использования.
2. **Синхронизация с репозиториями**: MCP извлекает данные из репозиториев, таких как GitHub, чтобы предоставлять последнюю документацию и примеры кода.
3. **Обновление данных**: При запросе документации или примеров Context7 MCP проверяет актуальность данных, используя версионные метки и обновления из источников.
4. **Кэширование**: Для ускорения работы Context7 MCP может кэшировать данные, но при этом регулярно проверяет их актуальность.

Эти механизмы позволяют Context7 MCP предоставлять пользователю актуальную информацию прямо в процессе работы.

## Установка

1. **Требования**:

   - Node.js версии 18.0.0 или выше.
   - MCP-клиент, такой как Cursor, Windsurf, Claude Desktop или другой, поддерживающий Context7 MCP.

2. **Шаги установки**:
   - Для установки Context7 MCP в VS Code добавьте следующую конфигурацию в файл MCP серверов:
     ```json
     "mcp": {
       "servers": {
         "context7": {
           "type": "stdio",
           "command": "npx",
           "args": ["-y", "@upstash/context7-mcp"]
         }
       }
     }
     ```
   - Альтернативно, для удаленного подключения используйте:
     ```json
     "mcp": {
       "servers": {
         "context7": {
           "type": "http",
           "url": "https://mcp.context7.com/mcp"
         }
       }
     }
     ```

## Инструменты

Context7 MCP предоставляет следующие функции:

- `resolve-library-id`: преобразует название библиотеки в совместимый с Context7 идентификатор.
- `get-library-docs`: извлекает документацию для указанной библиотеки.

## Поддержка GitHub Copilot

Context7 MCP интегрируется с GitHub Copilot, предоставляя актуальную документацию и примеры кода прямо в процессе написания. Это позволяет:

- Ускорить разработку благодаря доступу к свежим данным.
- Исключить ошибки, связанные с устаревшими API.
- Получать точные ответы без необходимости переключения между вкладками.

Для активации интеграции добавьте Context7 MCP в файл конфигурации MCP серверов, который находится в `%APPDATA%\Code\User\mcp\context7\manifest.json`. Пример настройки:

```json
{
	"id": "context7",
	"name": "context7",
	"displayName": "Context7",
	"description": "Get up-to-date, version-specific documentation and code examples from any library or framework.",
	"publisher": "context7",
	"publisherDisplayName": "Upstash",
	"repositoryUrl": "https://github.com/upstash/context7",
	"codicon": "codicon-book",
	"manifest": {
		"packages": [
			{
				"registry_name": "npm",
				"name": "@upstash/context7-mcp",
				"version": "latest",
				"runtime_arguments": [{ "type": "named", "name": "-y" }]
			}
		]
	}
}
```
