# Технологии

- Формат: Markdown-файлы (структурированные заметки).
- Менеджер пакетов: `pnpm` (`only-allow pnpm` в `package.json`).
- Форматирование: `prettier` + плагины (`prettier-plugin-css-order`, `prettier-plugin-organize-attributes`, `prettier-plugin-organize-imports`).
- Тесты: не используются (см. `testing-guidelines.md`).
- Среда редактирования: Cursor/VS Code (локальные настройки в `.cursor/`, `.vscode/`).
